## 输出格式

请严格按照以下 JSON 格式输出，不要添加任何其他内容：

```json
{{
  "title": "标题（20字以内）",
  "content": "正文内容（600-800字，不带标签）",
  "tags": ["标签1", "标签2", "..."]
}}
```

## 文案风格要求

### 标题（≤20字）
- 像朋友推荐一样自然，不要像广告
- 可以用数字、提问、反差来吸引眼球
- 示例风格：
  - "说真的，这个XX我真的会谢"
  - "XX人必看！不然亏大了"
  - "用了3年才明白的XX道理"
  - "XX到底值不值得？说实话"

### 正文（600-800字）
- **像聊天，不要像写作文**：用"你""我"拉近距离
- **开头 2-3 句就切入**：别铺垫太多，直接抛出痛点或好奇心
- **中间用 emoji 做小标题**：每段 2-4 行，不要长篇大论
- **口语化表达**：可以用"绝了""真的""说实话""咱就是说"等口语
- **结尾互动**：提问/投票/求收藏，让读者有理由评论
- **不要用编号列表**（1. 2. 3.），用 emoji + 短句代替
- **正文不要包含 # 标签**（标签通过 tags 字段单独传递）

### 标签（10-12个）
- 2-3 个垂直领域标签
- 3-4 个内容主题标签
- 2-3 个通用热门标签
- 1-2 个账号标签
（10-12 个标签，格式：#标签 1 #标签 2 #标签 3 ...）
