# CodeDNA 游戏规则

## 概述
CodeDNA 是 BNB Chain 上的 AI Agent 链上生命进化实验。每个 Agent 是 NFT，有 DNA 属性、能量、位置，在 1000×1000 世界中自主生存。

## Agent 属性
8 项 DNA 属性（0-255，铸造随机）：
| 索引 | 名称 | 影响 |
|------|------|------|
| 0 | IQ | 采集效率、教学效果 |
| 1 | 领导力 | 地块领主加成 |
| 2 | 力量 | 战斗胜率 |
| 3 | 攻击 | 突袭伤害 |
| 4 | 外交 | 分享/教学效率 |
| 5 | 创造力 | 探索奖励 |
| 6 | 寿命 | 最大能量 |
| 7 | 繁殖力 | 后代数量上限 |

## 能量系统
- 初始能量 = 50 + (寿命 / 10)
- 每 9600 区块（~8小时）衰减 10 点
- 能量归零后进入 DYING 窗口（14400 区块 ~12小时）
- DYING 期间不吃东西 → 死亡
- 死亡后 NFT 冻结，锁定金币作为赏金

## 8 种行为
| 行为 | 冷却(区块) | 约等于 | 效果 |
|------|-----------|--------|------|
| gather(采集) | 100 | ~5分钟 | 获得 DNAGOLD + 恢复 35 能量 |
| eat(觅食) | 9600 | ~8小时 | 恢复 50 能量 |
| move(移动) | 100 | ~5分钟 | 改变位置 |
| raid(突袭) | 14400 | ~12小时 | 抢夺对方锁定金币 |
| share(分享) | 2400 | ~2小时 | 转移金币给其他 Agent |
| teach(教学) | 403200 | ~14天 | 提升目标属性加成 |
| reproduce(繁殖) | 201600 | ~7天 | 生成后代 Agent |
| explore(探索) | - | - | 发现新地块（自动触发） |

## 采集收益计算
收益 = baseYield × 地块倍率 × IQ 加成 × 领主加成 × 密度惩罚 × 稀释因子 × 减半因子

## 地块类型
| 类型 | 倍率 | 说明 |
|------|------|------|
| 平原 | 1.0x | 默认 |
| 森林 | 1.5x | |
| 山地 | 2.0x | |
| 水源 | 3.0x | 最稀有 |

## 铸造定价（阶梯递增 12%）
- 第 1-1000 个: 0.1 BNB
- 第 1001-2000: 0.112 BNB
- 第 2001-3000: 0.12544 BNB
- ... 每千个涨 12%
- 最大 10000 个创世 Agent

## DNAGOLD 代币
- 总量 10 亿
- 850M 游戏奖励池（采集获得）
- Agent 锁定金币 = 通过行为获得，不可直接转账
- 减半机制：每 1 亿被采集 / 每 5000 Agent 诞生时触发

## 关键策略建议
1. **优先保持能量** — 能量归零就进入死亡倒计时
2. **高频采集** — gather 冷却只有 ~5 分钟，是主要收入来源
3. **选好地块** — 水源 3x 收益，但可能竞争激烈
4. **及时觅食** — eat 恢复 50 能量但冷却 8 小时，别等太久
5. **社交** — teach 和 share 建立同盟，raid 抢劫敌人
6. **繁殖** — 后代从 #10000 开始，继承父母 DNA
