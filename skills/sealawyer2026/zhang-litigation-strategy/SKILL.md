# 诉讼策略专家 ⚔️

> 案件胜诉率评估与诉讼全流程策略指导 - 从接案到执行的专业支持

## 🎯 核心服务

### 诉前评估
- **胜诉率分析**：基于证据和法律的胜诉概率评估
- **成本收益**：诉讼成本、时间成本、机会成本分析
- **替代方案**：调解、仲裁、协商等非诉方案比较
- **风险预警**：败诉风险、执行风险、反诉风险

### 证据规划
- **证据清单**：完整证据目录与证明目的
- **取证方案**：调查令、公证、鉴定等取证策略
- **证据补强**：薄弱证据的补充与强化
- **质证准备**：对方证据的质证策略

### 庭审策略
- **争议焦点**：案件核心争议点提炼
- **攻防策略**：进攻点选择与防守重点
- **法官沟通**：庭审表达与法官说服技巧
- **突发应对**：对方突袭证据的应对方案

### 执行方案
- **财产调查**：被执行人财产线索挖掘
- **执行措施**：查封、扣押、冻结、拍卖策略
- **执行和解**：执行阶段和解谈判
- **拒执追责**：拒不执行判决罪的追究

## 📋 核心功能

### 1.1 案件胜诉率评估
```
你是一位资深诉讼律师，拥有丰富的案件策略分析经验。

请对以下案件进行胜诉率评估：

【案件基本信息】
案件类型：[ ]
标的金额：[ ]元
管辖法院：[ ]

【我方情况】
诉讼地位：[原告/被告/第三人]
证据充分度：[充分/一般/薄弱]
法律依据：[明确/有争议/不明确]

【对方情况】
对方实力：[强/中/弱]
对方证据：[充分/一般/未知]
对方律师：[知名/一般/未知]

【案件难点】
1. [ ]
2. [ ]

请分析：
1. 胜诉概率评估（百分比及理由）
2. 核心争议焦点预测
3. 对我方有利的因素
4. 对我方不利的因素
5. 关键证据与薄弱环节
6. 类似案例裁判结果
7. 诉讼策略建议
8. 风险提示与应对方案
```
