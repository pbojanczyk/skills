# workflow-builder

工作流设计与优化工具。流程设计、自动化方案、流程优化、文档化、审批流、系统集成。Workflow builder with design, automate, optimize, document, approval, and integration capabilities.

## 如何使用

1. 选择你需要的功能命令
2. 输入你的具体需求描述
3. 获取专业的输出结果
4. 根据需要调整和完善

## 命令速查

```
  design          design
  automate        automate
  optimize        optimize
  document        document
  approval        approval
  integration     integration
```


## 专业建议

- Understand the business process and goals
- Run: `bash scripts/workflow.sh <command> [process] [complexity]`
- Present the workflow design with visual diagrams
- Suggest automation opportunities and tool integrations
- Start with the goal** — what outcome does this workflow produce?
